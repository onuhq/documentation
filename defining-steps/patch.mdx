---
title: "onu.v1.http.patch"
description: "Makes a PATCH request"
---

### Example

```yaml
main:
  name: Send a demo PATCH request
  input:
    name:
      type: string
  steps:
    - name: Make a PATCH request
      uses: onu.v1.http.patch
      with:
        url: https://demo.joinonu.com/api/say-hello
        body:
          name: ${input.name}
      result: requestResult

    - name: Return result
      return: ${requestResult.body}
```

### Parameters

<ResponseField name="url">
  Required. The URL to send the PUT request to
</ResponseField>

<ResponseField name="headers">
  Optional. Any headers to add to this request. This is represented by key value
  pairs
</ResponseField>

<ResponseField name="body">
  The body of the request. This is represented by key value pairs
</ResponseField>

### Response

Returns an object with `status_code`, `body` and `headers`
