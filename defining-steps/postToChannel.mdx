---
title: "onu.v1.slack.postToChannel"
description: "Posts a message to a Slack channel"
---

<Note>
**AUTH REQUIRED**

You must connect to Slack on the
[Integrations](https://app.joinonu.com/settings#integrations) page before using
this step

</Note>

### Example[](#example "Direct link to Example")

```yaml
main:
  name: Send a demo Slack Message
  input:
    message:
      type: string
    channelName:
      type: string
  steps:
    - name: "Post to #beta channel in slack"
      uses: onu.v1.slack.postToChannel
      with:
        channelName: ${input.channelName}
        markdown: >
          ${input.message + "\nSent via Onu"}
        botUsername: Onu Bot
        botImageUrl: >
          https://res.cloudinary.com/onudev/image/upload/v1677447405/Onu/purple-light_cdjl3l_1_uyvon8.png
```

### Parameters

<ResponseField name="channelName" required>
  The name of the Slack channel to post to
</ResponseField>

<ResponseField name="markdown" required>
  The message to post
</ResponseField>

<ResponseField name="botUsername"=>Optional. The name of the bot</ResponseField>

<ResponseField name="botImageUrl">Optional. The image URL to use for the Slack bot profile photo</ResponseField>

### Response

Returns an object with `success`
