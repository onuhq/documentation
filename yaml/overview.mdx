The first iteration of Onu allowed you to define tasks using a YAML configuration file.  
While we still support tasks configured in YAML, we suggest you migrate to one of our SDKs for more flexibility.


If you are looking for a more guided introduction, checkout the [getting started guide](/introduction/getting-started).
