# Install
```bash
yarn add @onuhq/node  # or npm i --save @onuhq/node
```

# Setup
In your project, create a new directory called `onu/`. This directory will contain all of your Onu tasks.
```
.
└── Project root/
    └── src/
        ├── services/
        ├── routes/
        └── onu/
            ├── index.ts
            ├── myCustomTask.ts
            └── myOtherCustomTask.ts
```


`onu/index.ts`

```typescript
import { OnuClient } from '@onuhq/node';


const onuClient = new OnuClient({
  onuPath: __dirname,
  apiKey: <YOUR_API_KEY>,
});


export default onuClient;
```
